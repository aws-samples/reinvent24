# KUB313 | Architecture Patterns for MLOps on Amazon EKS
Amazon Elastic Kubernetes Service (Amazon EKS) helps organizations tackle machine learning operations (MLOps) challenges by providing a highly secure, reliable, scalable, and flexible Kubernetes environment in the cloud. In this chalk talk, explore architecture patterns for training, fine tuning, and serving machine learning models using Amazon EKS, Amazon EC2, and AWS storage services.
